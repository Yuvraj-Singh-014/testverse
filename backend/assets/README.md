# Certificate Template

## Instructions

Save the certificate template image as:
```
backend/assets/certificate_template.png
```

## Template Specifications

- **Format**: PNG
- **Orientation**: Landscape
- **Recommended Size**: 1754 × 1240 pixels (A4 landscape at 150 DPI)
- **Alternative Size**: 2480 × 1754 pixels (A4 landscape at 300 DPI)

## Name Positioning

The participant's name will be automatically overlaid:
- **Font**: Times-Bold
- **Size**: 36pt
- **Color**: Black (#000000)
- **Position**: Centered horizontally, at 56.5% from the top
- **Alignment**: Center

This positions the name on the horizontal line in the certificate template.

## Current Template

The current template is for the **HackuVerse AI/ML Quiz Mania** certificate with:
- nerds room, HackuVerse, and NodeZer0 logos at the top
- GeeksforGeeks logo
- "CERTIFICATE OF PARTICIPATION" title
- Fixed event description text
- Date and seal at the bottom

The name will be dynamically inserted on the horizontal line above "has successfully participated in".
